1.1 Basic Ruby Language literacy

Learn syntax, Write Code, Run programs

Meet Interactive Ruby (irb), your new best friend

  1. irb --simple-prompt

A Ruby syntax survival kit

Assignment: 

x=1
  • = binds a local variable on the left to an object on the right.

Print something on the screen: 

puts x

print x

p x
  • p outputs and inspect string which may contain extra info about what it's printing

Get a line of keyboard input: 

string = gets

hello!

print string
  • gets assigns the input line directly to a variable.

Convert a numeric string to a number: 

x="100".to_i
  • arithmetic won't work with numeric strings

Compare to Values: 

x == y

Conditional execution: 

if x==y

	puts "true"

else

	puts "false"

end
  • Conditional statements always end with the word end

Special value objects: 

true

nil

false
  • nil is a kind of "non-object"
    • indicates the absence of a value or result

      *false and nil cause a conditional statement to fail

  • NB0 and empty strings cause a conditional statement to succeed!

The default object: 

self
  • Self is a role that different objects play, depending on the execution context

Comments: 

\#A Comment

1.1.3 The variety of Ruby identifiers

Ruby has a small number of identifier types that you'll want to be able to recognise, helps with understanding code

Local Variables Always start lowercase or underscore 

x, string, __abc__, start_value

Instance Variables Serve the purpose of storing information for individual objects 

@age, @last_name


NB - Doesn't start with Uppercase

Class Variables Same as above 

@@running-total

Global Variable No rules, so long as it has the $ sign

Constants Can start with UpperCase, CamelCase i.e. FirstName or FIRST_NAME

Keywords

  • About 40 of them, and short, i.e:
    • def (method definitions)
    • class (class definitions)
    • if (conditional execution)
    • `FILE' (Name of file currently being executed

Method Names

  • Same as local variables except they end with ?, ! or =
  • They don't want to draw attention to themselves
  • are simply expressions that provide a value
